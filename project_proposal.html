Project Detail 2.1 Project Goals: The purpose of the project is to cater the needs of those individuals in search for job and those entities which provide jobs. It caters the need of both the entities, building a platform for connecting and communication. Employers can as well list their job and view all the applicants for the specific job. In this way the employers and the applicants can connect in a professional platform to full fill their requirements. The platform is a web based portal functioning on any web browsers such as Google Chrome, Mozilla Firefox etc. 2.2 Project Scope: , this portal will have following three main and the important features mentioned below:

Admin Options

  • Admin is able to handle all the following operations
  • Admin can view jobs
  • Admin can view users
  • Admin can view companies
  • Admin can delete a job
  • Reservation System
  • Admin can delete a user
  • Admin can delete a company

User Options

  • User can sign up new account
  • User can login.
  • User can view all jobs
  • User can apply for a job
  • User can attach CV to a job
    • User can generate a CV

User Options

  • Companies can sign up new account
  • Companies can post a job along with all the details and requirements.
